# data_final

Hier darf öffentlich nur der finale anonymisierte Datensatz liegen:

- `final_analysis_dataset_anonymized.csv`
- `final_analysis_dataset_anonymized.rds`
- optional: `final_analysis_dataset_anonymized_export_summary.csv`

Vor einem GitHub-Release muss geprüft werden, dass keine direkten Identifikatoren, E-Mail-Adressen, Response-IDs, IP-Adressen, Namen oder Koordinaten enthalten sind.
